# Sleep Solutions India

A professional website redesign for Sleep Solutions India — a premier sleep medicine clinic.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion (animations)
- shadcn/ui components
- Wouter (routing)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy on Vercel

Click the button or run:

```bash
npm run build
```

Then deploy the `dist/` folder to Vercel. SPA routing is handled by `vercel.json`.

## Pages

- `/` — Home (hero, services, testimonials, CTA)
- `/services` — Services offered
- `/conditions` — Sleep disorders treated
- `/doctors` — Meet the team
- `/contact` — Contact & appointment form
